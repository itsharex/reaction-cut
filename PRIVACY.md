# Privacy Policy

This application runs locally. It does not operate a remote server.

## Data stored locally

- Login credentials or cookies (if you log in)
- Task metadata (downloads, clips, uploads, workflow state)
- Logs and diagnostics

## Storage locations (macOS)

- App data: `~/Library/Application Support/com.tbw.reaction-cut-rust/`
- Database: `bili-clip-flow.sqlite3`
- Logs: `app_debug.log`, `auth_debug.log`, `panic_debug.log`

## Network access

The app communicates with Bilibili endpoints and optional Baidu Netdisk endpoints
to download/upload content. No additional telemetry is sent by default.

## Removing data

To remove all local data, delete the app data directory listed above.
